Known Remaining Differences

Architectural (By Design)

  • post_debug returns JSON (Python returns HTML)
  • add_feed returns JSON (Python returns 302 redirect)
  • search_feeds uses PodcastIndex API (Python uses iTunes)

Pipeline (Deferred)

  • No ad segment merging/filtering before audio cut
  • No heuristic fallback when LLM boundary refinement fails
  • No token rate limiting
  • No max_completion_tokens for newer OpenAI models
  • No pre-reprocess snapshot creation
